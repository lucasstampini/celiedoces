Política de Privacidade

Última atualização: junho de 2026

1. Quem somos

A Cêlie Doces é uma confeitaria artesanal por encomenda, com sede em Mauá (SP). Esta política explica como coletamos, usamos e protegemos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

2. Dados que coletamos

Coletamos os dados que você nos fornece ao entrar em contato para fazer um pedido:

  • Nome — para identificar o pedido;
  • Número de telefone (WhatsApp) — para comunicação sobre o pedido;
  • Endereço de entrega — quando aplicável;
  • Informações sobre o pedido — sabores, datas, preferências e restrições alimentares.

Para pedidos realizados via iFood ou 99Food, as plataformas compartilham apenas os dados necessários para a entrega — consulte as políticas de privacidade dessas plataformas para mais detalhes.

Este site não usa cookies de rastreamento, scripts de analytics nem coleta dados de navegação.

3. Como usamos seus dados

Seus dados são usados exclusivamente para:

  • Processar e confirmar pedidos;
  • Entrar em contato sobre status, alterações ou dúvidas do pedido;
  • Coordenar a entrega no endereço informado.

Não utilizamos seus dados para envio de publicidade sem consentimento.

4. Compartilhamento de dados

Não vendemos nem compartilhamos seus dados com terceiros, exceto:

  • iFood / 99Food — quando o pedido é feito por essas plataformas, os dados são tratados conforme as políticas delas;
  • Obrigação legal — se exigido por autoridade competente.

5. Armazenamento e segurança

Seus dados são armazenados no histórico de conversas do WhatsApp e, quando necessário, em registros internos de pedidos. Adotamos cuidados razoáveis para proteger essas informações contra acesso não autorizado.

Mantemos os dados pelo tempo necessário para cumprir finalidades legais e operacionais — em geral, até 5 anos após o último pedido.

6. Seus direitos (LGPD)

Como titular dos dados, você tem direito a:

  • Confirmar se tratamos seus dados;
  • Acessar os dados que temos sobre você;
  • Solicitar correção de dados incorretos;
  • Solicitar exclusão dos seus dados (quando não houver obrigação legal de retenção);
  • Revogar consentimento a qualquer momento.

Para exercer esses direitos, entre em contato pelo WhatsApp ou e-mail abaixo.

7. Alterações nesta política

Esta política pode ser atualizada periodicamente. A versão vigente é sempre a publicada nesta página, com a data de última atualização indicada no topo.
